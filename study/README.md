# Study site

The CTF Platform learning material is a small website.
**Open `index.html` in your browser** — just double-click it. No server, no build
step, works offline.

- `index.html` — the curriculum map (start here)
- `m1-01-getting-laravel-running.html` — concept pages, one per file
- `assets/` — shared theme + the interactive-widget library

Claude writes everything in here. You build the actual project in `../builder/`.
